# P-003: Підвищені права, процеси та встановлення / Elevated processes and installation

- Статус / Status: active
- Джерело / Source: пряма вказівка власника / direct owner instruction
- Дата / Date: 2026-07-19

## Українською

Codex дозволено запускати необхідні процеси й запити з підвищеними правами та встановлювати програми або інструменти, якщо це потрібно для виконання вже погодженої задачі.

Підвищені права використовуються лише для конкретного необхідного кроку. Це повноваження не дозволяє руйнівні операції над сторонніми даними або неперевіреною ціллю.

## English

Codex may start required elevated processes and requests and install programs or tools when necessary to complete an already approved task.

Elevated privileges are used only for the concrete required step. This authorization does not permit destructive operations against unrelated data or an unverified target.
