# Приватний дослідницький архів для Versie 1

Приватний репозиторій [Tarasevych/gmail-telegram-onderzoeksarchief](https://github.com/Tarasevych/gmail-telegram-onderzoeksarchief) зберігає доказову історію, на якій базується `Versie 1`: точні видимі події зашифровані, а рішення, проблеми, перевірені результати й відкриті питання подані у санітизованих двомовних реєстрах.

## Правила

- Усі нові виправлення продукту вносяться лише до активної `Versie 1`, доки власник прямо не накаже створити наступний випуск.
- Архів використовується для provenance та неповторення помилок, але не для автоматичного повернення старого коду або deployment.
- Точний ciphertext відкривається лише локально з owner-only DPAPI key-reference.
- Жоден відновлений секрет або конфіденційний payload не переноситься у Git, GitHub UI, issue, log чи повідомлення.
- Нові архівні сегменти є append-only і мають перевірятися за хешами та readback.
- Hidden reasoning, system prompts і developer prompts не експортуються; зберігаються видимі повідомлення та перевірюваний action/evidence trace.
