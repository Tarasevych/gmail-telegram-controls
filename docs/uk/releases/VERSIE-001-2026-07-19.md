## Source-продовження: owner-only Advanced Gmail reads

- `REQ-0024` додає protected allowlisted adapter для owner-викликів `messages.list`, `messages.get` і `history.list`.
- External Gmail connections зберігають connection-scoped direct HTTP і ніколи не успадковують Apps Script owner identity; mutations та unsupported reads також лишаються на direct HTTP.
- Adapter вимкнений, доки protected property `GMAIL_OWNER_ADVANCED_READ_V1` не дорівнює точно `enabled`. Жодні property, deployment, trigger, Telegram menu, OAuth record або Gmail data не змінено.
- Advanced Service errors передаються без hidden HTTP fallback. Adapter-тести проходять 8/8; повний suite має 451/452, бо exact immutable v57 hash gate навмисно відхиляє змінений source. Цю ізольовану branch не можна merge або deploy, доки власник окремо не дозволить наступний immutable; зменшення quota та live delivery лишаються неперевіреними.
- Production лишається immutable v55; immutable v56 лишається історією, а єдиний owner-only staging v57 збережено.
