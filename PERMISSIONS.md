# Повноваження / Permissions

Ця гілка містить лише повноваження, прямо надані власником проєкту. Вона не є документацією продукту, журналом релізів або місцем для припущених дозволів.

## Правило роботи

1. Спочатку знайти потрібну тему в індексі нижче.
2. Прочитати й змінити лише один релевантний файл.
3. Не завантажувати, не узагальнювати й не змінювати інші записи, якщо запит їх не стосується.
4. Змінювати повноваження лише за новою прямою вказівкою власника.
5. Якщо тема нова, створити окремий файл; не додавати різні повноваження в один запис.
6. Не зберігати паролі, токени, cookies, ключі, recovery-коди або способи обходу захисту.
