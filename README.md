# Gmail Telegram Controls

Top-level Telegram Mini App for `@TarasevychGmailNotifierBot`.

The page contains no Gmail credentials, bot token, chat ID, or permanent
control key. Telegram supplies signed `initData` at runtime, while the private
button key is carried only in the individual URL generated by Apps Script.
The Apps Script backend always validates that 256-bit key and additionally
validates signed `initData` whenever the Telegram launch mode supplies it.

Mailbox mutations (`archive`, `trash`, `spam`, and `unsubscribe`) always show
an explicit confirmation screen and require both the private key and fresh,
owner-bound Telegram `initData`. The delete control moves mail to Gmail Trash;
it never performs permanent deletion.
